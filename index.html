Sobre Mim

Sou desenvolvedor front-end com ampla experiência em HTML, CSS (incluindo Sass e Tailwind CSS), e JavaScript. Atualmente, estou expandindo minhas habilidades aprendendo React. Já realizei diversos projetos e landing pages que alcançaram mais de 90 pontos no PageSpeed, garantindo alta performance e ótima experiência do usuário.

Meu foco é criar sites rápidos, responsivos e visualmente atraentes, sempre seguindo as melhores práticas de desenvolvimento web. Tenho um portfólio variado que inclui desde pequenas landing pages até aplicações web mais complexas, todas otimizadas para SEO e com design moderno.

Além das minhas habilidades técnicas, sou apaixonado por aprender e acompanhar as tendências mais recentes em desenvolvimento front-end. Se você precisa de um site que não só atenda às suas necessidades, mas também supere suas expectativas, sou a pessoa certa para te ajudar. Vamos trabalhar juntos para transformar suas ideias em realidade!

Epic Burgers Landing Page

Epic Burgers é um site fictício para um restaurante especializado em hambúrgueres gourmet. O projeto foi desenvolvido como parte de um estudo de front-end utilizando HTML, CSS, JavaScript e Tailwind CSS para estilização.

Cafeteria Landing Page

Este é um projeto de site para uma cafeteria fictícia, desenvolvido usando HTML e Tailwind CSS.

Food Peek Landing Page

landing page do projeto Food Peek, uma plataforma para amantes da gastronomia. A página é responsiva e foi desenvolvida usando HTML5, CSS3, Tailwind CSS e JavaScript.

Gerador de Conselhos

Um aplicativo da web que gera conselhos aleatórios a cada clique. Construído usando HTML, CSS e JavaScript, este aplicativo demonstra habilidades de desenvolvimento web e integração de API.

Aplicativo de Clima

Um aplicativo meteorológico que busca e exibe as informações meteorológicas atuais de uma cidade específica usando a API OpenWeatherMap. Os usuários podem inserir o nome de uma cidade para obter dados meteorológicos em tempo real, incluindo temperatura, descrição do tempo e umidade.

One Piece Landing Page

Um projeto interativo apresentando personagens populares do anime "One Piece". Desenvolvido em HTML, CSS e JavaScript, permite aos usuários explorar informações sobre cada personagem. Perfeito para fãs da série e entusiastas de desenvolvimento web.
